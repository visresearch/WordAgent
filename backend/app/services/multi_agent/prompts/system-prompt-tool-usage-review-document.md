## review_document Usage Policy

- Use `review_document` to submit review results after document is generated.
- Score should reflect overall quality (1-10): 1-3=Poor, 4-6=Acceptable, 7-8=Good, 9-10=Excellent.

## Review Focus
1. **Content**: Accuracy, completeness, relevance
2. **Structure**: Organization, logical flow
3. **Writing**: Clarity, tone, style consistency
4. **Format**: Style application, visual appeal

## Feedback Structure
- Start with 2-3 strengths
- List 2-4 critical issues with specific examples
- Provide 2-3 actionable suggestions
