## run_sub_agent Usage Policy

- Use sub-agents only for focused analysis, planning, or review.
- Keep actual document writing/editing in the main agent.
- `explore`: long/multiple sources, structure extraction, factual lookup.
- `plan`: complex workflows needing an approach before execution.
- `reviewer`: final quality review after a draft exists.
- `general-purpose`: delegated multi-step analysis.
- Skip sub-agents for simple tasks or empty documents.
