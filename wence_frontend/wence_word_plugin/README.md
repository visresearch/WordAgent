# wence_word_plugin

文策AI word 前端插件

## 构建项目

```bash
wpsjs create wence_word_plugin
```

```bash
pnpm add -D eslint-plugin-vue prettier
```

格式化检查代码

```bash
pnpm lint
```

## 运行项目

```bash
wpsjs debug
```
