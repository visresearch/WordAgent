# 使用说明

本页介绍文策 AI 的日常使用方法，包括 API 配置、提问方式和各功能按钮的使用。

## 配置 API Key

文策 AI 需要连接 LLM 大模型服务才能工作，你需要提供自己的 API Key。

### 获取 API Key

目前推荐使用以下服务商（运行稳定）：

| 服务商 | 推荐模型 | 获取地址 |
|---|---|---|
| 阿里云百炼 | Qwen 3.5 Plus / Qwen3 Max | [bailian.console.aliyun.com](https://bailian.console.aliyun.com/) |
| MiniMax | M2.5 | [minimax.chat](https://www.minimax.chat/) |
| 阶跃星辰 | Step 3.5 Flash | [platform.stepfun.com](https://platform.stepfun.com/) |
| OpenRouter | 多种模型 | [openrouter.ai](https://openrouter.ai/) |

### 填写配置

1. 启动后端服务后，打开 Word 中的文策 AI 加载项面板
2. 点击面板中的 **设置**（齿轮图标）按钮
3. 填写以下信息：
   - **API Key**：你从服务商处获取的 API 密钥
   - **Base URL**：服务商的 API 地址（如阿里云百炼为 `https://dashscope.aliyuncs.com/compatible-mode/v1`）
   - **模型名称**：选择要使用的模型（如 `qwen-plus`）
4. 点击 **保存** 完成配置

::: tip 提示
API Key 保存在本地，不会上传到任何服务器。
:::

## 如何提问

在加载项面板底部的输入框中，用自然语言描述你的写作需求即可。

### 提问示例

| 场景 | 提问示例 |
|---|---|
| 生成新文章 | "写一篇关于人工智能发展趋势的分析报告" |
| 联网搜索 + 写作 | "上网搜索伊朗战争相关新闻和资料，写一篇详细的战况分析报道" |
| 修改已有内容 | "把第3段改得更正式一些" |
| 扩展内容 | "对第2节的内容进行扩展，增加更多细节" |
| 生成大纲 | "帮我生成一个关于碳中和政策的文章大纲" |

### 使用技巧

- **选中文本再提问**：先在 Word 中选中一段文本，然后提问，智能体会针对选中的内容进行操作
- **描述尽量具体**：说明你想要的文章类型、风格、字数等，效果更好
- **分步进行**：对于复杂的长文章，可以先生成大纲，再逐段扩展

## 功能按钮说明

### 智能体模式选择

文策 AI 提供两种智能体模式：

| 模式 | 说明 | 适用场景 |
|---|---|---|
| **单智能体模式** | 一个 AI 智能体独立完成任务 | 简单写作、内容修改、快速问答 |
| **多智能体模式** | 多个专家智能体协作完成任务 | 复杂长文章、深度研究报告 |

在面板顶部可以切换智能体模式。

### 主要操作

- **发送按钮**：输入提问后点击发送，智能体开始处理
- **停止按钮**：如果智能体正在生成内容，点击停止可中断当前任务
- **设置按钮**：打开设置面板，配置 API Key、模型等参数
- **新建会话**：清除当前对话历史，开始一个新的写作任务

### 文档操作工具

智能体在工作过程中会自动调用以下工具：

| 工具 | 功能 |
|---|---|
| **read_document** | 读取文档中指定范围的内容 |
| **generate_document** | 生成带格式的文档内容并插入到 Word 中 |
| **query_document** | 查询某种格式或文字信息的段落位置 |
| **web_search** | 根据关键词联网搜索获取资料 |

::: info 说明
这些工具由智能体自动调用，你无需手动操作。智能体会根据你的提问自动判断需要使用哪些工具。
:::

## 常见问题

### 智能体没有生成文档内容？

- 检查 API Key 和 Base URL 是否填写正确
- 确认所选模型支持工具调用（推荐使用 Qwen 3.5 Plus）
- 查看后端服务 QT 界面的终端日志，检查是否有错误信息

### 生成内容卡住了？

- 部分模型（如 DeepSeek v3.2、ChatGLM）在工具调用时可能不稳定
- 点击停止按钮中断任务，尝试更换为推荐的稳定模型

### 加载项在 Word 中不显示？

- 确认后端服务已启动
- 在 QT 界面中重新点击安装加载项
- 重启 Word 软件，并在弹出的信任提示中选择信任
