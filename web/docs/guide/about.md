## 文策AI（WordAgent）

文策 AI 是一个开源的 AI 辅助写作系统，以 WPS 和 Microsoft Word 加载项形式呈现，旨在让普通用户无门槛获得优质的 AI 写作辅助体验。

## 项目信息

- **项目地址**：[GitHub - WordAgent](https://github.com/visresearch/WordAgent)
- **开源协议**：[Apache License 2.0](https://www.apache.org/licenses/LICENSE-2.0)

## 开发计划

- [x] 支持 WPS Word 桌面客户端
- [x] 支持 Windows、Linux
- [x] 支持单智能体模式
- [x] 支持 Token 消耗优化
- [x] 支持多智能体模式
