# 启动 Microsoft Word 加载项

## 前提条件

- 已完成 [安装方式](/guide/installation) 中的后端服务部署
- 拥有 Microsoft 365 账户并可使用 Word 网页版或桌面客户端版

## 使用方式

Microsoft Word 加载项以在线模式运行，通过 HTTPS 连接本地后端服务。你可以按下列任一方式使用。

### 1. 启动后端服务

双击运行 `wence_ai` 可执行文件（或通过源码启动），确保后端服务已正常运行。

![](/MSoffice_pyQt.png)

### 2. 在安装界面准备加载项资源

在后端服务 QT 界面的 Microsoft Word 页面中：

1. 点击 **下载 manifest.xml**，将文件保存到本地。
2. 点击 **启动 HTTPS 服务**。
3. 点击 **用浏览器打开**，在浏览器中接受本地自签名证书（首次可能使用需要）。

### 3. 配置 Word 网页版加载项（可选）

1. 打开 [Word 网页版](https://word.cloud.microsoft/) 并登录 Microsoft 365 账户。
2. 进入：**开始 → 加载项 → 更多加载项 → 我的加载项 → 管理我的加载项 → 上传我的加载项**。
3. 选择并上传第 2 步下载的 `manifest.xml`。
4. 上传后如果侧边面板未立即显示，请刷新页面。

![](/MSoffice_web.png)

### 4. 配置 Word 客户端加载项（推荐）

1. 将下载好的 `manifest.xml` 放到一个空文件夹中。
2. 右键该文件夹，打开 **属性 → 共享 → 共享**，选择 **Everyone** 后点击 **共享**，记录网络路径。
   
![](/share_folder.png)

3. 打开 Microsoft Word 客户端，进入：
	**文件 → 选项 → 信任中心 → 信任中心设置 → 受信任的加载项目录**。
4. 在“目录 URL”中填入上一步的网络路径，点击添加目录，然后重启 Word。

![](/trusted_center.png)

5. 若重启后仍未显示加载项，进入：
	**文件 → 选项 → 自定义功能区**，将“开发工具”添加到主选项卡。

![](/custom_ribbon.png)

6. 在 Word 顶部打开 **开发工具**，点击 **加载项 → 共享文件夹 → 文策 AI 助手**。

![](/load_addin.png)

### 5. 开始使用

加载项出现后即可在侧边栏使用文策 AI。使用前请确保后端服务和 HTTPS 服务保持运行。
