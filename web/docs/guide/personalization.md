# 个性化配置

在设置面板的 **个性化** 标签页中，你可以调整 AI 的写作风格与回复行为，使其更符合你的使用习惯。

## 自定义指令

![](/user_config1.png)

自定义指令是一段**全局提示词**，会在每次对话开始时自动注入，帮助 AI 更好地理解你的写作偏好。

### 设置方法

1. 打开设置面板，切换到 **个性化** 标签页
2. 在 **全局提示词** 文本框中输入你的要求
3. 点击 **保存设置** 生效

### 快速模板

如果你不知道从何写起，可以直接点击预置的快速模板，一键填入常用的风格指令：

| 模板 | 说明 |
|---|---|
| **学术写作** | 专业、严谨的学术风格，适合论文、报告等正式场合 |
| **创意写作** | 富有想象力的创作风格，适合故事、散文等内容 |
| **商务文档** | 简洁、专业的商务风格，适合邮件、方案、报告 |
| **日常交流** | 轻松、友好的对话风格，适合日常沟通场景 |

点击任意模板会将其填入文本框，你也可以在此基础上继续修改。

### 示例指令

```
用中文回答我的问题，回复要简明扼要，避免废话。
```

```
写作时注重逻辑结构，每个段落聚焦一个核心观点，段落间用自然的过渡语句衔接。
```

::: tip 提示
自定义指令越简洁越好，过长的指令反而可能导致 AI 优先级混乱。建议不超过 100 字。
:::

## LLM 温度

![](/user_config2.png)

温度（Temperature）控制 AI 输出的**创造性与随机性**，数值范围为 0 到 2。建议默认0.7，适合大多数写作场景。

### 三种模式

| 模式 | 温度范围 | 特点 | 适合场景 |
|---|---|---|---|
| **精确模式** | 0 – 0.5 | 输出确定、一致 | 事实性写作、数据报告、技术文档 |
| **平衡模式** | 0.5 – 1.5 | 准确性与创造性兼顾 | 大多数写作场景（**推荐**） |
| **创意模式** | 1.5 – 2 | 输出随机、富有创意 | 头脑风暴、创意写作、广告文案 |

### 调整方法

1. 在 **LLM 温度** 区域拖动滑块，或点击三种模式之一快速设置
2. 滑块右上角实时显示当前数值
3. 点击 **保存设置** 生效

::: warning 注意
创意模式（>1.5）下 AI 输出较为随机，可能出现不符合事实的内容，不建议用于需要准确信息的写作任务。
:::
